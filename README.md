# BeOnline
BeOnline can be used to simulate moving of mouse pointer without physically moving the mouse.
This will be helpfull when you are not the admin of the system and you were not allowed to set the screen timeout to automatically lock the system.

# How to use

1. Clone the project
2. Compile the class using javac 
3. Run the compiled code.
4. You will see a popup with the details to follow.
